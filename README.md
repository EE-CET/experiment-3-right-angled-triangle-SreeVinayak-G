# Experiment 3: Right-Angled Triangle

## Problem Statement

In this task, you need to print a right-angled triangle of height 3 (lines) and base 5 (stars).

## Input Format

No input.

## Output Format

Print the triangle of height 3 and base 5 as shown.

### Example 1

**Input:**

```text
No input
```

**Output**

```text
*
* * *
* * * * *
```

**Explanation**
The shape printed is a right-angled triangle.

* Row 1: 1 star.

* Row 2: 3 stars.

* Row 3: 5 stars.

This creates a triangle with a height of 3 lines and the last line (base) having 5 stars.
